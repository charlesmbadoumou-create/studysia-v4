# EduScroll Africa Prototype

## Installation

```bash
npm install
npm run dev
```

## Structure

- `index.html`: point d'entrée HTML
- `vite.config.js`: config Vite
- `tailwind.config.js`: config Tailwind
- `postcss.config.js`: config PostCSS
- `src/main.jsx`: bootstrap React
- `src/App.jsx`: prototype UI basé sur ton code
- `src/index.css`: directives Tailwind + styles globaux
